Redirection

What is meant by redirection?

Redirection means being directed to another place on the web. There are several ways to achieve this.

The anchor tag in HTML

The anchor tag is commonly used for redirecting to another URL. This can be another endpoint or an entirely different website. 

<a href="https://www.google.com/">Take me awaaaaay</a>

This is a simple and effective method, but there are also more dynamic ways to handle redirection using JavaScript.

The global variable 'window'

window is the global object representing the browser window in which your script is running. Since everything runs within this context, everything becomes a property of the window object.

This means that: 

const currentLocation = window.location.href;

is the same as: 

const currentLocation = location.href;

location.href

Setting location.href to a new value navigates the window to that page. 

location.href = 'https://www.google.com/';

This will redirect the user to google.com and add the current page to the browser history, allowing the user to navigate back using the back button.

location.replace

Using location.replace() also redirects to a new page. 

location.replace('https://www.google.com/');

However, this method does not add the current page to the browser history, meaning the back button will not return the user to the previous page.
