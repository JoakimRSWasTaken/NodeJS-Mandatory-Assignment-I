XSS

What is XSS?

XSS, or Cross-Site Scripting, is a way for a hacker to inject malicious JavaScript into your website. This usually happens when a user's input is not handled, or sanitized, thoroughly before being run in the browser. An example could be the user submitting the text: 

<script>alert("Hacked!")</script>

This script would then be executed in the browser if the text was not validated beforehand.

Try googling the case of car license plates containing the text 'DROP TABLE'-something. It is a very serious security issue but also extremely funny to me.

How to avoid XSS

Escape or Sanitize

You can convert special characters into other, safer characters before rendering. 

const safeInput = req.body.name.replace(//g, ">");

This turns the contents of req.body.name into a safe string by replacing < with &lt; and > with &gt;, turning the earlier example into harmless text.

Use safe methods for what you want to do

You can use res.send() for JSON and res.sendFile() for static files.

Use Libraries to do all this for you

Packages like xss and sanitize-html exist for this purpose to make your life easier.
