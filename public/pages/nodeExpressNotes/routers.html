Routers

What are Routers?

Routers are a way to collect a bunch of routes. And what are routes? Here is an example of a route: 

get('/hello', (req, res) => {
    res.send({ data: "Hello!" });
});

These routes take two arguments: an endpoint as a string (/hello in this example) and a callback function with req and res defining what to do with the request and response.

What would your app.js look like if you had 10-20 of these routes? Messy! Correct! That is why we use Routers. We divide the routes into separate files to keep things organized, then import them into app.js.

Example Router

import { Router } from 'express';

const router = Router();

import { frontpagePage, aboutPage, contactPage } from "../util/pagesUtil.js";

router.get('/', (req, res) => {
    res.send(frontpagePage);
});

router.get('/about', (req, res) => {
    res.send(aboutPage);
});

router.get('/contact', (req, res) => {
    res.send(contactPage);
});

export default router;

Note how we import the named export Router from express and instantiate it. At the end of the file we export it, and then we import it into app.js: 

import express from 'express';

const app = express();

app.use(express.static('public'));

import exampleRouter from "./routers/exampleRouter.js";

app.use(exampleRouter);

const PORT = process.env.PORT || 8080;

const server = app.listen(PORT, () => {
    console.log('Server is listening on port', PORT);
});
