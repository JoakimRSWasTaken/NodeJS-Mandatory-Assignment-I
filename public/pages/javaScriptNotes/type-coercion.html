Type Coercion

What is type coercion?

Type coercion is the process of converting a value into a data type that it was not originally. Type coercion happens automatically and is usually done when performing operations that require specific data types.

Examples of type coercion

A very simple example can be the line 


console.log(1 + '2');

where the engine running the JavaScript coerces the string '2' into the number 2.

Type coercion can also happen when using comparison operators: 


console.log(null !== 0);

To be able to use the comparison operator !== (or the inverse ===), we need the two arguments to be the same data type. In this example they are both converted to simple boolean expressions.

The expression printed from this line of code is: 


false
