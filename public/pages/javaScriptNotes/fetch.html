Fetch

What is fetching?

Fetching in JavaScript is simply put a way to fetch data from an API endpoint or similar sources. Fetch requests make use of promises to handle their asynchronous nature.

Their asynchronous nature comes from the time it takes to perform the parallel operations in your code versus the time it takes to receive the data you are fetching. A promise refers to a promise of data being delivered.

Below is an example of how to make a proxy endpoint for redirecting to google.com: 


app.get('/googleproxy', (req, res) => {
  fetch('https://www.google.com/')
    .then((response) => response.text())
    .then((result) => {
      res.send(result);
    });
});

In this example we have a GET request to an endpoint called /googleproxy on a server we have running.

First it fetches google.com.

Then we get the response from Google. This is stored as HTML in the response variable.

With the response.text() method we convert the HTML into a string.

Finally we send the HTML content back as a response to the client accessing the endpoint.
