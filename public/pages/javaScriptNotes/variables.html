Variables

What are variables?

Variables are an invaluable tool that lets you store data in different forms and use it again later in your code. They help increase readability and reduce duplicated code fragments, keeping your code DRY (Don't Repeat Yourself).

Defining variables in JavaScript

We have more than one choice when choosing which keyword to use when assigning a variable. We have:

  • var
  • const
  • let

var

Var is the oldschool keyword for defining variables. 

var petDog = "Faxi";

There are some issues with var. Variables assigned with var will have their values kept through the scope they are defined in, "bleeding through" the scope.

const

Const is a more recent addition. It lets you assign a value to your variable and does not let you overwrite this value afterwards. It is recommended to default to const unless you know you want to change the value later. 

const otherPetDog = "Rökkvi";

let

Let is also a newer addition. It lets you assign a value while allowing you to change it later. 

let petCat = "Sammy";
