Functions

What are functions?

Functions are a way to modularize your code by splitting it up into reusable parts. These reusable parts are called functions in JavaScript.

First Class Citizen

Functions are first class citizens in JavaScript. This means that they have the same status as the other data types. Functions can be passed as arguments to other functions and be returned from functions, just like any other data type.

Regular functions

You define regular functions with the function keyword: 

function sayHello(name) {
  console.log('Hello ' + name);
}

They can be called again from anywhere in the same scope due to the fact that they are being hoisted to the top of the call stack.

Anonymous functions

Anonymous functions are functions without a defined name and are often used as callback functions. 

const sayHello = function(name) {
  console.log('Hello ' + name);
};

They are not used that much anymore and instead we often use the next type of function.

Arrow functions

Arrow functions are an easy way to write functions directly where they are supposed to be used. These are also commonly used as callback functions. 

const sayHello = (name) => {
  console.log('Hello ' + name);
};

They will be widely used.

Callback functions

Callback functions are functions that are passed to another function as an argument. This happens quite often and the example below is something you will see frequently. 

app.get('/', (req, res) => {
  res.sendFile('frontpage.html');
});
